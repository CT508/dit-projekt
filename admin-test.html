Non-negotiable matching rules

  • Match only when normalized feed GTIN/EAN equals approved master_products.ean.
  • SKU/MPN C13T44J240 is useful metadata, but it must never match the product.
  • Product title can be displayed, searched, and audited, but it must never attach an offer to a master product.
  • If EAN is missing, invalid, duplicate, or not approved, the row is rejected and shown in import_errors.
